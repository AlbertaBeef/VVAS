~~~~~~~~~~~~~~~~~~~~~
VCU Decoder Plug-in
~~~~~~~~~~~~~~~~~~~~~
“ivas_xvcudec” is GStreamer Plugin for hardware accelerated H264/H265 Video decoding using VCU. 
“ivas_xvcudec” plug-in does decoding of H264/H265 streams using VCU decoder soft-kernel running on ARM connected to host via PCIe. This plug-in accepts input encoded stream in byte-stream/NALU format only and produces NV12 frames based VCU frame alignment requirements.
To achieve zero-copy for higher performance, decoder plug-in does not bring data from device to host. Instead, it enables flag IVAS_SYNC_FROM_DEVICE on GstIvasMemory which holds raw NV12 frame. In a given GStreamer pipeline, if any of the following element (e.g. filesink) after VCU decoder would like to read and/or write on decoded frame, then DMA copy will be triggered by IVAS allocator.
